# Sales Prediction Using Machine Learning

## Project Overview

This project predicts product sales based on advertising expenditures using Machine Learning.

The objective is to analyze how advertising investments in different media channels influence sales and build a predictive model using Linear Regression.

## Dataset

The dataset contains advertising expenditures across different channels:

* TV Advertising Budget
* Radio Advertising Budget
* Newspaper Advertising Budget
* Sales

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-Learn

## Machine Learning Algorithm

* Linear Regression

## Project Workflow

1. Data Collection
2. Data Exploration
3. Data Visualization
4. Correlation Analysis
5. Feature Selection
6. Train-Test Split
7. Model Training
8. Prediction
9. Model Evaluation

## Visualizations

* Correlation Heatmap
* Scatter Plot
* Actual vs Predicted Sales Graph

## Results

* Successfully built a Linear Regression model.
* Predicted sales based on advertising expenditures.
* Evaluated model performance using R² Score.
