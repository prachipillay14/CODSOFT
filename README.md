# CodSoft Data Science Internship Projects

Welcome to my Data Science Internship project repository completed under the CodSoft Internship Program.

This repository contains Machine Learning projects developed using Python, Pandas, NumPy, Matplotlib, Seaborn, and Scikit-Learn.

## Projects Included

### 1. Iris Flower Classification

* Classification of Iris flowers into different species.
* Machine Learning Algorithm: K-Nearest Neighbors (KNN)
* Skills Learned:

  * Data Analysis
  * Data Visualization
  * Classification
  * Model Evaluation

### 2. Titanic Survival Prediction

* Prediction of passenger survival based on demographic and travel information.
* Machine Learning Algorithm: Logistic Regression
* Skills Learned:

  * Data Cleaning
  * Handling Missing Values
  * Classification
  * Feature Selection

### 3. Sales Prediction Using Machine Learning

* Prediction of product sales based on advertising expenditures.
* Machine Learning Algorithm: Linear Regression
* Skills Learned:

  * Regression Analysis
  * Correlation Analysis
  * Data Visualization
  * Predictive Modeling
