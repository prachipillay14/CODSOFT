# Titanic Survival Prediction

## Project Overview
This project predicts whether a passenger survived the Titanic disaster using Machine Learning.

## Dataset
Titanic passenger dataset containing information such as:
- Age
- Gender
- Passenger Class
- Fare
- Siblings/Spouse
- Parents/Children

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-Learn

## Machine Learning Algorithm
- Logistic Regression

## Project Workflow
1. Data Loading
2. Data Cleaning
3. Handling Missing Values
4. Data Visualization
5. Feature Selection
6. Model Training
7. Model Evaluation
8. Prediction
